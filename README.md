# 📦 Sistema de Inventário

Aplicação desenvolvida em C# para gerenciamento de estoque, permitindo o controle de produtos, quantidades e informações armazenadas em banco de dados SQLite.

---

## 🚀 Funcionalidades

* 📥 Cadastrar novos produtos
* 📋 Listar produtos em estoque
* ✏️ Atualizar informações dos produtos
* ❌ Remover produtos
* 🔍 Buscar produtos específicos
* 📊 Controle de quantidade em estoque

---

## 🛠️ Tecnologias utilizadas

* C#
* .NET
* SQLite

## 🧠 Como funciona

O sistema utiliza um banco de dados SQLite para armazenar os dados dos produtos.
Através de um menu interativo no terminal, o usuário pode:

1. Cadastrar produtos com nome, quantidade e preço
2. Visualizar todos os itens cadastrados
3. Atualizar informações dos produtos
4. Remover produtos do estoque
5. Buscar produtos específicos
